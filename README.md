# ymate-vscode-extension
ymate extension for vscode

## 主要功能

  1. java工程目录类视图
  2. 解析Ymate工程结构
  3. properties文件unicode中文编辑支持

## 目录结构

  1. 类视图
  2. 实体类
  3. 服务类
  4. 控制器
  5. 拦截器
  6. 验证器
  7. 存储器
  8. 配置项

安装开发依赖
```
npm install
```
安装打包工具（已安装过请忽略）
```
npm install -g vsce
```
打包插件安装文件 vsix
```
vsce package
```
